# VANA Browser Agent

VANA Browser Agent is an advanced AI-powered browsing agent designed to navigate the web freely and efficiently.  
It is capable of browsing websites, viewing web content and videos, performing advanced searches (including ticket searches), and moving seamlessly between multiple websites like a real user.

The agent provides a flexible and intelligent browsing experience suitable for automation, research, and assisted web navigation.

---

## Overview

VANA Browser Agent allows users to:
- Browse websites without restrictions
- View and interact with web pages
- Watch online videos
- Perform advanced searches
- Search for tickets and online services
- Navigate automatically between different websites
- Use AI-assisted decision making during browsing

## API Configuration (Required)

VANA Browser Agent requires an API key from **Google AI Studio**.

Steps:
1. Visit Google AI Studio.
2. Generate a valid API key.
3. Open the VANA Browser Agent application.
4. Paste the API key into the **API** section inside the application.
5. Click **Save**.
6. Restart the application if necessary.
